# Currency Converter - Week 1 Assignment

A simple Python currency converter that converts Nigerian Naira (NGN) to multiple international currencies.

## Description

This program allows users to convert amounts from Nigerian Naira (NGN) to three major international currencies: US Dollar (USD), Euro (EUR), and British Pound (GBP). The converter provides real-time conversion using predefined exchange rates and displays results in a clear, formatted manner.

## Features

- ✅ Convert NGN to USD, EUR, and GBP
- ✅ User-friendly input prompts
- ✅ Formatted output with currency symbols
- ✅ Error handling for invalid inputs
- ✅ Clean, well-structured code using functions and dictionaries

## Requirements

- Python 3.x
- `rich` library for enhanced terminal output (Optional, Just for pretty formatting of outputs, probably an overkill for this assignment)


## Usage

1. Navigate to the Week-1-assign directory
2. Run the program:
   ```bash
   python currency_converter.py
   ```
3. Enter the amount in Nigerian Naira when prompted
4. View the converted values in USD, EUR, and GBP
